# Aditya Sharma – Developer Portfolio

A modern, interactive personal portfolio website designed to effectively showcase my technical skills, projects, and experience. This project emphasizes smooth animations, high performance, and a refined dark-mode user interface.

## Overview

This portfolio serves as a central platform to present my work as a front-end developer. It combines visually engaging elements with a clean and responsive layout to deliver an optimal user experience across all devices.

## Created By

**Aditya Sharma** - Developer & AI/ML Enthusiast

## Key Features

- **Interactive User Interface**  
  Smooth page transitions and scroll-based animations enhance user engagement.

- **Command Palette Navigation**  
  Keyboard shortcut (`Ctrl + K` / `Cmd + K`) enables quick and efficient navigation across the website.

- **Custom Animations**  
  Includes a dynamic particle background rendered using canvas and a fully customized interactive cursor.

- **Spotlight Card Effects**  
  Subtle hover-based spotlight animations on experience and education sections for improved visual appeal.

- **Terminal-Inspired Design**  
  "Terminal-style" components used in About and Skills sections to create a distinctive developer-centric aesthetic.

- **Responsive Design**  
  Fully optimized for mobile, tablet, and desktop devices to ensure consistent usability.

## Tech Stack

- **Frontend Framework:** React 18 (with Vite)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
